# Template Thumbnails

This folder contains thumbnail images for PowerPoint templates displayed on the results page.

## File Format

- **Filename**: `{template_id}.png`
- **Size**: 200x150px recommended (4:3 aspect ratio)
- **Format**: PNG with transparency support

## Required Thumbnails

Create thumbnail images for each template:

- `professional.png` - Professional template thumbnail
- `modern.png` - Modern template thumbnail  
- `minimalist.png` - Minimalist template thumbnail
- `creative.png` - Creative template thumbnail
- `educational.png` - Educational template thumbnail

## Creating Thumbnails

1. Open your PowerPoint template file
2. Take a screenshot of the first slide
3. Crop and resize to 200x150px
4. Save as PNG with the template ID as filename
5. Place in this folder

## Fallback

If a thumbnail image is not found, a gray SVG placeholder will be displayed automatically.
